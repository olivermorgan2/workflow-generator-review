# Templates

Starter templates for the artifacts that the kit generates into a new target
project. Each template is a markdown file with placeholders; skills fill the
placeholders when they run.

Templates are copied into target projects as-is or transformed by a skill. They
are not used directly inside this kit repo beyond reference examples.
